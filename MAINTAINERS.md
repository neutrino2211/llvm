# Maintainers

This repository is a community-maintained fork.

No official transfer of maintainership for upstream `llir/llvm` has been announced.

## Ownership

- Fork maintainers: project owners of this repository and active contributors.
- Contributions are accepted via pull requests.

## Current priorities

1. Update LLVM IR compatibility to latest stable LLVM release.
2. Keep CI green on supported Go versions and platforms.
3. Cut a new release once upgrade work is stable.

## Contact

- Use GitHub issues for bugs, regressions, and feature requests.
- Use pull requests for proposed changes.
